## 👤 Autor

* **Módulo:** Entornos de Desarrollo

## 🛠️ Tecnologías y Herramientas
* **Lenguaje:** Java ☕

## 📂 Estructura del Proyecto
* `/src`: Código fuente original y refactorizado (`CalculadoraPedido.java`).
* `/doc`: Documentación técnica generada automáticamente en HTML.
* `MEMORIA.md`: Documento detallado con las capturas de pantalla y la justificación de cada refactorización aplicada.

## 🚀 Puntos destacados del trabajo
1. **Configuración Profesional:** Repositorio con reglas de protección de rama y flujo de Pull Requests.
2. **Código Limpio (Clean Code):** Aplicación de patrones para eliminar números mágicos y mejorar la modularidad.
3. **Manual Técnico:** Javadoc completo integrado en el repositorio.
